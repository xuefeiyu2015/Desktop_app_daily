# Project: Desktop To-Do Widget

Build a minimal desktop to-do widget using Python.

Requirements:
- Use customtkinter
- Semi-transparent floating window
- Always on top
- Clean modern UI
- Rounded corners
- Soft shadow style

Features:
1. Add todo item
2. Delete todo item
3. Mark completed
4. Reminder popup
5. Save data to todo.json

UI requirements:
- Glassmorphism style
- Semi transparent background
- Minimalist font
- Small floating widget

Structure:

main.py
- start the application

ui/widget_ui.py
- build the UI

core/todo_manager.py
- handle todo logic
- load/save JSON

Use good Python structure.
Keep code clean and modular.
